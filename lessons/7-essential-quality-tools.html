The investigation sequence

1. FlowchartUnderstand the actual process and possible origin points.
2. Check SheetCollect consistent, analysis-ready evidence.
3. StratificationSeparate mixed populations before concluding.
4. ParetoFocus on the vital few categories.
5. FishboneGenerate testable causal hypotheses.
6. HistogramStudy centering, spread, shape, and mixtures.
7. Scatter PlotTest whether a suspected X moves with Y.
8. Control ChartVerify stability and sustainment over time.

Important terminology note

Published lists differ. Some include stratification; others substitute a flowchart or run chart. This lesson teaches all eight practical capabilities because omitting stratification or process mapping creates serious investigation risk.

Core rule: choose the tool from the question you need answered—not from habit, software availability, or the chart you already know.
