Learning objectives

By the end of this lesson, you will be able to:

  1. Select the tool that matches the investigation question.
  2. Explain the inputs, outputs, and limitations of each tool.
  3. Interpret common patterns without overstating causation.
  4. Sequence the tools in a realistic quality investigation.

The investigation sequence

Each tool answers a different question. The output of one stage determines the next move.

1. FlowchartWhere can the issue originate?
2. Check SheetHow will evidence be collected?
3. StratificationWhich subgroup carries the issue?
4. ParetoWhat deserves priority?
5. FishboneWhat could cause it?
6. HistogramWhat is the distribution doing?
7. Scatter PlotDo X and Y move together?
8. Control ChartIs the process stable and did the fix hold?

Summary

  • Map before collecting data.
  • Stratify before interpreting pooled results.
  • Pareto prioritizes but does not prove cause.
  • Fishbone branches are hypotheses until validated.
  • Control charts distinguish common and special causes.
Decision rule: use the least-complex tool that answers the current unresolved question with defensible evidence.
