Boundary rule: use the basic QC tools while the problem still needs evidence. Use these planning tools once the cause is credible and the team must execute a response.

Driver logic

If “unclear procedure” drives both training inconsistency and equipment misuse, Procedure has more outgoing influence and becomes a leverage point.

Count arrows as a structured discussion aid—not as a statistical measure of causal strength.

PDPC challenge

Plan: add a temperature alarm.

What could go wrong? nuisance alarms create alarm fatigue.

Countermeasure: validated alarm limits, escalation logic, calibration checks, and response training.

PDPC is plan-risk analysis. It is not a replacement for process FMEA where a formal risk method is required.

Knowledge check

The activity network reveals that the deadline is impossible. What is the disciplined response?

Common failure modes

  • Predefining affinity categories.
  • Stopping the tree one level too early.
  • Assuming ownership is obvious.
  • Skipping PDPC as “negative thinking.”
  • Scheduling before the task list is complete.
